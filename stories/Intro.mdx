# Button

Элемент, нажатие на который приводит к какому-либо ответному действию (реагированию).

При необходимости ознакомьтесь с [исходным кодом](https://github.com/2e32/react-button) компонента.

## Начните работать

В storybook представлены материалы, позволяющие ознакомиться с функционалом кнопки.

_Свойства (props)_:

- изучите список свойств, которые принимает компонент
- изменяйте значения свойств, чтобы увидеть какой эффект они дадут
- экспериментируйте со свойствами в [Playground](?path=/story/example-button-props--playground)

_Использование (usage)_:

- ознакомьтесь с расширенным применением кнопки (стилизация, события, использование в сетке). Некоторые примеры могут носить практический характер
